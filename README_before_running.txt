Scramble homework solution fo Ericsson by Tomolya Mátyás:
Made in Python language with doctest unit test.
For trying out the solution please read the instructions below:

If you are using Linux OS:
	- navigate in terminal to the downloaded scramble directory
	- make run.bat runnable (chmod 755 run.bat)
	- run it by typing ./run.bat in the directory
	- the test results are shown

If you are using Windows OS:
	- Please add the python interpreter to the global enviroment
	  PATH variable. Only then you are able to run the solution.
	- navigate in cmd to the downloaded scramble directory
	- run the run.bat file by typing run.bat
	- the test results are shown
